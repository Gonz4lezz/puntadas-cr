# ESPECIFICACIONES TÉCNICAS - PROYECTO PUNTADAS

## 1. TECNOLOGÍAS UTILIZADAS
- HTML5, CSS3, JavaScript
- Bootstrap 5.3.0
- Vue.js 3
- Google Maps API
- EmailJS

## 2. ESTRUCTURA DEL PROYECTO
- index.html (Home)
- pages/catalog.html (Catálogo con API)
- pages/contact.html (Formulario completo)
- pages/about.html (Historia)
- pages/location.html (Geolocalización)
- pages/author.html (Información del autor)
- data/products.json (API interna)

## 3. APIS IMPLEMENTADAS
- API Externa: Hacienda CR (tipo de cambio)
- API Interna: products.json
- Google Maps API
- EmailJS API

## 4. FUNCIONALIDADES JAVASCRIPT
- Validaciones de formulario (Vue.js)
- Filtrado dinámico de productos (Vue.js)
- Geolocalización y rutas (Vue.js + Google Maps)
- Animaciones y efectos (JavaScript vanilla)

## 5. RESPONSIVE DESIGN
- Mobile First approach

- Media queries personalizadas
